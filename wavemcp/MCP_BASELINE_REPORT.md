## 核心发现

1. **MCP 首次实现全面净正收益（+9）**，简单提示词策略优于强制使用
2. **cva6 最稳定**（+3），在所有提示词版本下都正收益
3. **Chisel 项目（rocketchip +4, xiangshan +6）提升最大**，说明 MCP 在复杂项目上也有帮助，关键是让 agent 自行决策
4. **ibex (-3) 和 caliptra (-1)** 有轻微退步，但相比旧提示词的 -7 和 -2 已明显改善
5. **无需复杂的负向约束提示词**，一句 "Use it when you need to" 效果最好
6. 所有项目 0 基础设施错误
